# Sales Intelligence Dashboard

Dashboard visualisasi data penjualan berbasis Streamlit.

## File yang dibutuhkan

- app.py
- requirements.txt
- data_penjualan.xlsx

## Cara menjalankan di Streamlit Community Cloud

1. Upload semua file ke repository GitHub.
2. Buka Streamlit Community Cloud.
3. Klik Create app.
4. Pilih repository GitHub.
5. Isi main file path: app.py.
6. Klik Deploy.
